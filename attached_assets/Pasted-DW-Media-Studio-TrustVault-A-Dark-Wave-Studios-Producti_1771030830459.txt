DW Media Studio (TrustVault)
A Dark Wave Studios Production
Part of the TrustLayer Ecosystem

What It Is:
DW Media Studio is a universal IP storage and media management platform built for creators, families, and teams. It serves as a secure digital vault where users can upload, organize, edit, and preview videos, audio, images, and documents -- all from one place. Designed as part of the larger Dark Wave Studios / TrustLayer ecosystem, it connects to blockchain identity tracking, cross-app single sign-on, and ecosystem-wide tools.

Media Editing Suite:

Image Editor -- Crop, rotate, flip, resize, filters, full color grading (brightness, contrast, saturation, blur, hue, temperature, vignette, sharpen), text overlays (draggable, font/size/color/bold), freehand drawing and annotation (brush, eraser, color, thickness), stickers and shapes (10 types, draggable, scalable)
Audio Editor -- Trim, fade in/out, volume control, 3-band EQ (bass/mid/treble), reverb, noise gate
Video Editor -- Trim, full color grading (brightness, contrast, saturation, hue, temperature, vignette), frame capture
Merge/Combine Tool -- Guided workflow for image collages, audio concatenation, video concatenation
All editors save results as new media items
